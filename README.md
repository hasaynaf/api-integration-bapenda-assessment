## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Start

Start server on http://localhost:3000

Step 1 :

```bash
npm run start
```

Step 2 : Call Api with CLI

```bash
npm run cli
```

Step 3 : Unit Test

```bash
npm run test
```
